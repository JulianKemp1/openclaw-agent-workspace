# 📋 Decisions Log — OpenClaw RAG System

> This file is maintained by the Manager Agent.
> All decisions requiring owner approval are logged here BEFORE action is taken.
> Owner signs off by writing ✅ APPROVED or ❌ REJECTED + reason.

---

## HOW TO USE THIS FILE

**Agents:** When you need owner approval, add an entry below using this format:
```
### [DECISION-XXX] Title
- **Date:** YYYY-MM-DD
- **Requested by:** [Agent Name]
- **Type:** Schema Change | Dependency | Architecture | Deployment | Other
- **Description:** What you want to do and why
- **Impact:** What breaks or changes if approved
- **Alternatives considered:** What else was considered
- **Status:** 🟡 PENDING
```

**Owner:** Review pending items and update Status to ✅ APPROVED or ❌ REJECTED.
